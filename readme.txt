Replaces default Howdy message with new Greeting depending on time of day

== Description ==

Replaces default Howdy message with new Greeting depending on time of day.
Good Morning, Good Afternoon, and Good Evening are the messages used. 

== Installation ==

1. Upload `greetadminbytime` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What timezone does this plugin use? =

This plugin uses the site local time as a basis for the greeting it displays
